06 • Requirements: The Code of Chivalry

Functional Requirements

  • ✦ Multi-agent AI scaffolding for real-time thought polishing.
  • ✦ Anonymous 'Safe-Zone' for initial drafting.
  • ✦ Facilitator dashboard with engagement heatmaps.
  • ✦ Real-time speech-to-text with translation support.

Non-Functional Requirements

  • ✦ Latency < 200ms for seamless dialogue flow.
  • ✦ High accessibility (WCAG 2.1 compliant).
  • ✦ Privacy-first data encryption for student thoughts.
  • ✦ Scalability for up to 100 concurrent seminar users.

"Playful" Must-Haves

  • ✦ Private Draft Sanctuary: Students draft privately, get AI polishing help, then choose when to publish to group discussion.
  • ✦ Balanced Participation Monitor: Teacher dashboard shows speaking distribution, silence duration, and underrepresented voices in real time.
  • ✦ Peer-First AI Prompting: AI nudges only when discussion stalls, and prompts peer interaction instead of replacing human dialogue.
